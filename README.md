### getting started
start using blitzy right now by visiting the [documentation](https://github.com/xerooup/blitzy2d/tree/main/documentation)<br>

### how to install:
`build.gradle.kts` example:
```kt
dependencies {
    implementation("io.github.xerooup:blitzy:VERSION")
}
```
or `pom.xml`:
```xml
<dependencies>
    <dependency>
        <groupId>io.github.xerooup</groupId>
        <artifactId>blitzy</artifactId>
        <version>VERSION</version>
    </dependency>
</dependencies>
```
